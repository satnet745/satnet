# SATNET GLOBAL Quick Start

SATNET GLOBAL is a local-first Android app that combines communication, maps, and SATNET tools in one launcher.

## Main areas

- **Communication**: Call, Messages, Contacts, Share Files, Connect, Help, and Share SATNET.
- **SATNET tools**: role setup, wallet setup/access, voucher flows, verifier tools, and merchant Lightning tools when enabled in the build.
- **Maps**: local-first maps, saved bookmarks, and Rhizome bookmark exchange.

## Current posture

- Android min SDK: 19
- Android target SDK: 34
- Release posture: pilot / pre-production
- Lightning, relay, and experimental routing are build-dependent
