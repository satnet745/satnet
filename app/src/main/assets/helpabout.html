About SATNET

SATNET is a local-first Android app for resilient communication and Bitcoin-enabled field tools. The current app combines mesh calling, messaging, contacts, file sharing, SATNET Maps, and role-based Bitcoin tools such as wallet access, voucher issuance/redemption, and verifier workflows where enabled in the build.
For more information, visit satnet.app.
Source code, release evidence, and documentation are maintained in the public repository.
