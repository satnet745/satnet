# SATNET GLOBAL - quick reference

## One-line summary

SATNET GLOBAL is a single Android app combining mesh communication, Bitcoin wallet tooling, vouchers, Maps, and build-dependent Lightning support.

## Main functions

- Calls
- Messages
- Contacts
- Share Files
- Connect / network controls
- Help
- Share SATNET
- SATNET tools
- Maps

## SATNET tools

- Role setup
- Bitcoin wallet setup and access
- Voucher issuance and redemption
- Verifier dashboard
- Merchant Lightning tools when enabled in the build

## Main roles

- User
- Agent
- Merchant
- Verifier

## Build facts

- Min SDK 19
- Target SDK 34
- Pilot / pre-production posture
- Lightning, relay, and experimental routing are feature-flagged

## Common commands

```bash
./gradlew assembleDebug
./gradlew test
./gradlew assembleRelease
```

## Safety reminders

- Not for emergency telephony.
- Hotspot and ad hoc networking may change device Wi-Fi behavior.
- Rhizome is distributed storage, not a private chat system.
